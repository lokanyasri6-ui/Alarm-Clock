# Alarm Clock ⏰

A simple Alarm Clock application developed using Python and Tkinter.

## Features

- Set an alarm time in HH:MM format
- User-friendly graphical interface
- Checks the current system time automatically
- Displays a popup message when the alarm time is reached

## Technologies Used

- Python
- Tkinter (GUI Library)
- Datetime Module
